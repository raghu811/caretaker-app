
# Caretaker Booking Website

## Project Overview
This project is a responsive mini website developed as part of a Website Development Internship assignment. The goal of this project is to demonstrate the ability to design and develop a simple caretaker booking platform interface using modern web development practices.

The website includes multiple pages that allow users to understand the service, explore features, and contact the service provider. The project focuses on clean UI design, responsive layout, and user-friendly navigation.

## Features
- Responsive design (Mobile, Tablet, Desktop)
- Clean and simple user interface
- Structured multi-page website
- Navigation between pages
- Contact form for user inquiries
- Consistent typography and spacing

## Pages Included
1. **Home Page**
   - Introduction to the caretaker booking service
   - Overview of services
   - Call-to-action section

2. **About / Services Page**
   - Information about caretaker services
   - Benefits of the platform
   - Key service highlights

3. **Contact Page**
   - Contact form for user inquiries
   - Contact details and location information

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Responsive Design (Flexbox / Grid)

## How to Run the Project
1. Download or clone the repository.
2. Open the project folder.
3. Open the `index.html` file in your browser.
